Test with Debug

📋 Expected Behavior:

  1. Click a test link above
  2. You should be redirected to the WordPress admin
  3. A new post editor should open with "ReBit" settings
  4. The ReBit URL block should be auto-inserted
  5. The URL field should be pre-populated with the shared URL
  6. Any title/text should appear in the post content area

🔍 Debugging Steps

  1. Check Browser Console: Open Developer Tools (F12) and look for messages starting with "BitStream:"
  2. Check WordPress Error Log: Look for messages containing "BitStream Share Debug"
  3. Test Direct Admin Link: Try going directly to /wp-admin/post-new.php?post_type=bit&rebit=1&shared_url=https://example.com
  4. Check PWA Installation: Make sure the BitStream PWA is properly installed on your device

📱 Real Android Share Test

If the tests above work, but sharing from the YouTube app doesn't:

  1. Open YouTube app on Android
  2. Find any video
  3. Tap the Share button
  4. Look for "BitStream" in the share menu
  5. If it's not there, the PWA might not be properly installed
  6. If it's there but doesn't work, check the browser console for errors
